Reference

API Documentation

Complete method reference for ColoX.

new Color(input)

Create a new Color instance from various input formats.

new Color("#ff0000") new Color(255, 0, 0) new Color("rgb(255,0,0)") new Color({r: 255, g: 0, b: 0})

color.plus(other)

Add two colors by averaging their RGB values.

const result = color1.plus(color2);

color.minus(other)

Subtract one color from another.

const result = color1.minus(color2);

color.mix(other, ratio)

Mix two colors with a ratio from 0 to 1.

const mixed = color1.mix(color2, 0.5);

color.lighten(percent)

Increase lightness by a percentage.

const lighter = color.lighten(20);

color.darken(percent)

Decrease lightness by a percentage.

const darker = color.darken(20);

color.getShades(count, spread)

Generate an array of shade variations.

const shades = color.getShades(5, 30);

color.getScale(count)

Generate a full lightness scale (10%-90%).

const scale = color.getScale(9);

Color.random()

Generate a random color with balanced saturation and lightness.

const random = Color.random();

color.toString(format)

Convert to string. Formats: hex, rgb, rgba, hsl, hsla, auto.

color.toString("hex") // #ff0000 color.toString("rgb") // rgb(255,0,0)
